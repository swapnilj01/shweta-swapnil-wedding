# Shweta weds Swapnil - Wedding Invite

A mobile-first GitHub Pages-ready wedding invite with:

- South Indian wedding inspired floral, lamp, garland, and temple motifs
- The couple photo integrated into a decorative invite frame
- Parallax scroll layers and soft reveal animations
- Live countdown to 23 December 2026, 8:00 PM IST
- RSVP mail link and Google Maps button
- Tap-to-play original browser-generated instrumental background ambience

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files from this folder to the repository root.
3. In GitHub, open Settings > Pages.
4. Under Build and deployment, select Deploy from a branch.
5. Choose the main branch and root folder, then save.
6. GitHub will publish the invite URL after the Pages build completes.

## Edit details

- Wedding text and links: edit `index.html`.
- Colors, spacing, flowers, lamps, and parallax visuals: edit `styles.css`.
- Countdown date and background music behavior: edit `script.js`.
- Couple photo: replace `assets/couple.webp` and `assets/couple.jpg` with your own images using the same filenames.
